# 📝 To-Do List
## Overview
The **To-Do List** is a simple web-based application that helps users organize and manage daily tasks efficiently. Users can add tasks, mark completed tasks, delete tasks, and save their task list using local storage so tasks remain available even after refreshing the page.

## Features
### Task Management
- Add new tasks instantly
- Display tasks in a list format
- Mark tasks as completed
- Delete tasks from the list

### Task Status
- Click on a task to mark it as completed
- Completed tasks are shown with a strike-through effect
- Visual indicators for completed and pending tasks

### Data Persistence
- Uses **Local Storage** to save tasks
- Tasks remain available even after refreshing the browser

### User Interface
- Clean and modern design
- Responsive layout
- Simple and user-friendly interface

## Technologies Used
- **HTML5** → Structure of the application  
- **CSS3** → Styling and layout design  
- **JavaScript** → Task functionality and local storage handling

## Software Requirements
- Any modern web browser (**Chrome, Edge, Firefox, etc.**)
- Code editor (**VS Code recommended**)

## How It Works

1. Enter a task in the input field
2. Click the **Add** button
3. The task is added to the task list
4. Click on a task to mark it as completed
5. Click the **X** icon to remove a task
6. Tasks are automatically stored in **Local Storage**

## How to Run the Project
1. Download or clone the repository
2. Open the project folder in VS Code
3. Run **index.html** in any browser
